search in history: 
    ctrl r -> use (destroy: ctrl q, execute: enter, back with command: esc)

    ctrl p, ctrl n

search file: locate -b (basename) -> search file named ... only

search multiple files: OR: locate doct docu (search doct or docu)
                               AND: locate -A doct docu (search all)
