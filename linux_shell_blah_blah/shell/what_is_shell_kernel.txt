    kernel (hat nhan, loi) is the core of linux os and it handles all 
 the software (application, package) to control your computer's 
 hardware (ex: cpu, rom, ram,...) and also handle the command you 
 input from terminal. There is many type of shell that exist

    terminal is a program that run a shell. In the past, terminal were
 moniters and keyboards but now is software. It is the input output inetrface 
 software (terminal emulator)

    shell (lop vo) is the softwares that let users communicate with 
 the kernel, it's the kernel's interface, the intermediate software 
 between user and kernel. It processes commands(send command to kernel
 and receive the result) and return the output.
